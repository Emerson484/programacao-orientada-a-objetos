# programacao-orientada-a-objetos
Este repositório contém exercícios e projetos desenvolvidos em Java durante minha graduação, com foco na aplicação prática dos princípios de Programação Orientada a Objetos (POO). Os códigos demonstram o uso de classes e objetos, encapsulamento, herança e polimorfismo, além de boas práticas de organização e estruturação.
